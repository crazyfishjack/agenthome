# Python开发环境配置指南

## 1. Python安装

### 1.1 Windows系统
1. 访问Python官网：https://www.python.org/downloads/
2. 下载Python 3.12或更高版本
3. 运行安装程序，勾选"Add Python to PATH"
4. 验证安装：打开命令行，输入`python --version`

### 1.2 macOS系统
1. 使用Homebrew安装：`brew install python`
2. 或从官网下载安装包
3. 验证安装：打开终端，输入`python3 --version`

### 1.3 Linux系统
1. Ubuntu/Debian：`sudo apt-get install python3`
2. CentOS/RHEL：`sudo yum install python3`
3. 验证安装：`python3 --version`

## 2. 虚拟环境管理

### 2.1 venv（Python内置）
```bash
# 创建虚拟环境
python -m venv myenv

# 激活虚拟环境
# Windows
myenv\Scripts\activate
# macOS/Linux
source myenv/bin/activate

# 退出虚拟环境
deactivate
```

### 2.2 conda
```bash
# 创建环境
conda create -n myenv python=3.12

# 激活环境
conda activate myenv

# 退出环境
conda deactivate
```

### 2.3 pipenv
```bash
# 安装pipenv
pip install pipenv

# 创建环境并安装依赖
pipenv install

# 激活环境
pipenv shell

# 退出环境
exit
```

## 5. 代码规范

### 5.1 PEP 8
- 使用4个空格缩进
- 每行不超过79个字符
- 使用空行分隔函数和类
- 导入语句放在文件顶部

### 5.2 类型提示
```python
from typing import List, Dict, Optional

def process_data(
    data: List[Dict[str, Any]],
    threshold: Optional[float] = None
) -> Dict[str, Any]:
    pass
```

### 5.3 文档字符串
```python
def calculate_sum(a: int, b: int) -> int:
    """
    计算两个整数的和

    Args:
        a: 第一个整数
        b: 第二个整数

    Returns:
        两个整数的和
    """
    return a + b
```

## 7. 性能优化

### 7.1 使用生成器
```python
# 不好的做法
def get_large_list():
    return [i for i in range(1000000)]

# 好的做法
def get_large_generator():
    for i in range(1000000):
        yield i
```

### 7.2 使用内置函数
```python
# 使用map、filter、reduce
result = map(lambda x: x**2, range(10))
result = filter(lambda x: x > 5, range(10))
```

### 7.3 缓存结果
```python

    # 昂贵的计算
```
